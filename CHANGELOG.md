# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-30

### Added
- Multi-source news collection (Eastmoney, Futu, Tonghuashun)
- Stock code + company name extraction
- Sector identification (10 sectors)
- Entity extraction (numbers, dates, policies, events)
- 7-step causal reasoning LLM prompt generation
- Structured Parquet output
- Signal list generation (buy/sell with confidence)
- 30 test cases
